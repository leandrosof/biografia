# 🚀 Portfólio Leandro Silva | Tech & Content Creator

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto é uma Single Page Application (SPA) desenvolvida para centralizar minha trajetória como Analista de Sistemas, Especialista em UX/IoT e Criador de Conteúdo.

🌐 **Status do Projeto:** Concluído / Em atualização contínua.

## 👨‍💻 Sobre o Projeto

Este site foi construído para ser o meu cartão de visitas digital. Ele mescla o meu lado desenvolvedor — focado em resolver problemas com código e interfaces responsivas — com o meu perfil autêntico de criador de conteúdo (@leandrosof), onde abordo tecnologia, lifestyle, rotina de treinos, humor e minhas gameplays de GTA RP e FIFA.

### 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica e otimizada para SEO.
* **CSS3:** Estilização responsiva, Dark Mode customizado, animações e layout em Grid/Flexbox.
* **JavaScript (Vanilla):** Interatividade, scroll suave e Intersection Observer para animações de revelação (fade-in).
